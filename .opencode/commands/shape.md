description: To have a better understanding of the application current state we will maintain uml diagrams in mermaidjs format
---

- Use git to get the current changes
- Only consider changes inside app/
- Identify what objects/entities are involved in the changes
- Add or update them in the mermaidjs file uml/entities.mmd
- Add or update a sequence diagram for each of the potentially affected routes in a mmd diagram inside uml/routes/
